* sor法解方程
```Matlab
function [x, n] = ssor(A, b, x0, w, eps)

%计算迭代矩阵

D = diag(diag(A));

L = -tril(A,-1);

U = -triu(A,1);

BS = (D-w*L)\((1-w)*D+w*U); %inv(M)*A=M\A

f = w*(D-w*L)\b;

%判断收敛性

if w>=2 || w<=0

disp('sor迭代不收敛');

return

else

n = 1;

x = BS*x0 + f;

while norm(x-x0,inf)>=eps

x0 = x;

x = BS*x0 + f;

n = n+1;

end

end
```

* 生成对称正定矩阵
```Matlab
function [A,b] = new(n) % ⽣成对称正定矩阵

b = randi(100,[n, 1]);

%ii = randi(100,[n, n]);

%A = ii*ii.';

M=diag(rand(n,1));

Z=orth(rand(n,n));

A=Z'*M*Z;

end
```
